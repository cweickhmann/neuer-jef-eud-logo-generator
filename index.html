Hinweis

Der Logo-Generator unterstützt aus verschiedenen technischen Gründen kein CMYK sondern nur RGB. Dies wird jedoch für viele Druckprodukte (zum Beispiel Flyer) benötigt. Um dieses Problem zu umgehen, hat sich bisher als praktisch gangbare Lösung folgendes Vorgehen bewährt:

  1. Benutzen Sie eine den Anforderungen (z.B. Hintergrundfarben) entsprechenden Variante des RGB-Logos.
  2. Importieren Sie die SVG-Datei in das Grafikprogramm (z.B. Scribus oder InDesign)
  3. Tauschen Sie die wenigen (üblicherweise 3 oder weniger) RGB-Farben durch ihre CMYK-Pendants manuell aus.

Dem Logo-Archiv, das der Logo-Generator erstellt, liegt daher eine Datei mit den korrespondierenden Farbwerten bei.

Note

For technical reasons, the Logo Generator only supports RGB and no CMYK. However, the latter is necessary for many print products (for example flyers). To work around this problem, the following procedure has so far proven practical:

  1. Use an RGB variant of the logo that suits the graphical requirements (e.g. background colour).
  2. Import the SVG file into the graphics programme (e.g. Scribus or InDesign)
  3. Exchange the few (typically 3 or less) RGB colours by their CMYK analogues.

The generator adds a file to the logo archive that lists the corresponding colour values.
