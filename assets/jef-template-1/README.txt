HINWEISE
========

Dieses Archiv wurde automatisch generiert und enthält Logodateien in
verschiedenen Ausführungen und Formaten.
Insbesondere die Dateien im bearbeitbaren Vektorformat SVG enthalten
nicht die notwendigen Schriften und lassen sich daher nur auf Systemen
korrekt anzeigen, bei denen diese installiert sind.

Daher sind diesem Archiv Schrift-Dateien begefügt, die zur Anzeige der Logos
notwendig sind. Bitte installieren Sie diese in der für Ihr System
notwendigen Art und Weise. Im Folgenden ein paar Hinweise, ohne Anspruch auf
Vollständigkeit oder Richtigkeit (insbesondere in der Zukunft):

Schrifteninstallation
=====================

Windows
=======

Wählen Sie die Schrift-Dateien mit Rechtsklick aus und wählen Sie in dem
sich öffnenden Menü "Installieren". Alternativ können Sie die Schriftarten
auch durch Doppelklick betrachten und dort auf "Installieren" klicken.
Weitere Informationen unter [1] oder z.B. [2].

MacOS
=====

Öffnen Sie die Schrift-Dateien durch Doppelklick. In dem sich öffnenden
Dialog, wählen Sie "Schrift installieren".
Weitere Informationen unter [3] oder z.B. [4].

Linux
=====

Die meisten grafischen Oberflächen (Gnome, KDE, Xfce, ...) haben ein
Werkzeug, das sich bei Doppelklick auf die Schrift-Dateien öffnet und einen
"Installieren"-Knopf anzeigt.
Alternativ können die Schriften-Dateien auch in das Verzeichnis
"~/.local/share/fonts" kopiert werden (wobei ~ dem Benutzerverzeichnis
entspricht) oder systemweit in "/usr/share/fonts". Im Anschluss sollte
`fc-cache -fr` ausgeführt werden, um den Schriften-Cache zu aktualisieren.
Weitere Informationen z.B. hier [5].

NOTES
=====

This archive has been automatically generated and contains logo files in
various styles and formats.
In particular, files in the editable SVG format do not contain the necessary
fonts and therefore can only be correctly displayed on systems with these
fonts installed.

Therefore, in this archive contains the font files necessary for correct
display. Please install these in the correct way for your system. The
following lines contain general instructions without any warranty that they
are correct (or are going to be in the future):

Font Installation
=================

Windows
=======

Choose the font file with a right-click and choose "Install" from the pop-up
menu. Alternatively, you can double-click the files and click on "Install"
in the font viewer application.
Further information under [1] or e.g. [2].

MacOS
=====

Open the font file with a double-click. The dialogue that now shows the font
also features a button that say "Install Font".
Further information under [3] or e.g. [4].

Linux
=====

Most graphical user interfaces (Gnome, KDE, Xfce, ...) come with a tool that
opens upon double-clicking a font file and shows a "Install"-button.
Alternatively, you can copy the font files into a directory named
"~/.local/share/fonts" (where ~ stands for the current user's home
directory) or system-wide into "/usr/share/fonts". Directly after that you
should run `fc-cache -fr` to update the font cache.
Further information under e.g. [5].
